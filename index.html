IMPORT TO PHONE CALENDAR

Download the .ics file then follow the steps for your device:

📱 IPHONE / APPLE CALENDAR
  1. Tap "Download .ics" below — the file will open automatically.
  2. Tap "Add All" when prompted — all events import into Apple Calendar.
  3. Open Calendar app and look for the new "Catholic Liturgical" calendar.
🤖 ANDROID / GOOGLE CALENDAR
  1. Tap "Download .ics" and save to your device.
  2. Open Google Calendar on desktop → Settings → Import & Export.
  3. Upload the .ics file and choose which calendar to add events to.
📧 OUTLOOK / MICROSOFT
  1. Download the .ics file to your computer.
  2. In Outlook: File → Open & Export → Import/Export → Import an iCalendar file.
  3. Select the downloaded file — events appear in your calendar.
